# 资源文件说明 (Resource Files Guide)

本目录包含程序的所有可视化资源文件，供下级开发商按需替换以自定义界面。

## 目录结构

```
res/
├── Bitmap/          # 菜单图标 (16x16 BMP, 灰色背景 RGB:240,240,240)
├── Cur/             # 光标文件
├── file/            # 文件管理器磁盘图标
├── 3rd/             # 第三方工具
├── *.ico            # 窗口/对话框图标
├── *.bmp            # 工具栏位图
└── *.rc2            # 资源包含文件
```

---

## 自定义指南

### 替换菜单图标
1. 准备 16x16 像素的 BMP 文件
2. 背景色使用 `RGB(240,240,240)` 以匹配对话框
3. 替换 `Bitmap/` 目录下对应文件
4. 重新编译程序

### 替换窗口图标
1. 准备 ICO 文件（建议包含 16x16、32x32、48x48 多尺寸）
2. 替换 `res/` 目录下对应文件
3. 重新编译程序

### 替换工具栏
1. 工具栏位图为多图拼接格式
2. 每个按钮图标宽高相同（主工具栏 48x48，文件工具栏 24x24）
3. 保持图标数量和顺序不变
4. 重新编译程序
